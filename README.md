# Celestiark Codex Package

このフォルダは、Codex にそのまま渡せる実装用パッケージです。

## 含まれているもの
- `SPEC.md` : Codex向け完全仕様
- `data/cards.json` : 実装用カードデータ
- `data/cards_source.json` : 元データ補助付きカードデータ
- `data/cards.csv` : 目視確認しやすいCSV
- `public/images/` : studentId で正規化した画像
- `public/videos/ssr.mp4`
- `public/videos/ur.mp4`
- `reference/` : 元のExcelと参照資料
- `RARITY_NOTE.md` : rarity / lore の補足
